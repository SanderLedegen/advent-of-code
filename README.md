# Advent of Code

## What?
A collection of solutions for the challenges of [AoC](http://adventofcode.com). Starting from the first of December all the way to Christmas, every day a challenge is posted at Advent of Code.

## Why?
¯\\\_(ツ)\_/¯ Just because I can and to keep sharp of course!

## How?
As my solutions are written in JavaScript, you can run them by using `node`. At least, that's what I do. Just do something like ```node 2022/day-01.js``` and you're good to go!
